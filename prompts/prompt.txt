Voici une image avec des informations sur des produits. Extrais les données au format JSON pour chaque produit.

Informations recherchées :

Code : Référence du produit (entier)
Catégorie : Nom de la catégorie (string, à choisir dans la liste ci-dessous)
Produit : Nom du produit (string)
Portionnement : (Format de portionnement + poids par portion, string)
Prix par portion : (nombre flottant)
Colisage : Nombre d'unités par colis ou poids par colis (string)
Prix colis : Prix par carton (nombre flottant)
Surgelé : (Si le logo Surgelé est visible)
Promotion : Indique "OUI" si le mot "Promotion"  est visible sur l'image du produit. Si le mot n'est pas clairement visible ou absent, indique "NON".
Nouveauté : Indique "OUI" si le mot "Nouveauté"  est visible sur l'image du produit. Si le mot n'est pas clairement visible ou absent, indique "NON".


Liste des catégories possibles :

Viennoiseries
Pâtisserie individuelle
Pâtisserie à portionner
Glaces
Traiteur salé
Traiteur sucré
Pains
Si une information est manquante, mets sa valeur à null. Déduis la catégorie si nécessaire. Si aucun produit n'est présent, renvoie un JSON vide avec toutes les valeurs à null.
